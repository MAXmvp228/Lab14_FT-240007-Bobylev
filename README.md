# Лабораторная работа №14 - Тесты
Язык программирования: Python.

Среда разработки: PyCharm2025.1.1.1.
# Инструкция по работе
1. Написать на любом языке программу тестирования. Тест представлен в исходном файле (структура описана ниже) - программа читает данные из файла (не встраивать вопросы в код)! Все вопросы в тесте подразумевают выбор одного правильного варианта.

Программа поочередно выводит вопрос и спрашивает ответ
Анализирует правильность ответа и выдает соответствующее сообщение с озвучиванием правильно ответа
В конце выдается количество правильных ответов из общего количества
Оценивается качество выполнения теста в процентах, при условии равнозначности всех вопросов (кол-во правильных ответов / общее количество вопросов в процентах)
Обеспечить функциональные и нефункциональные критерии качества программы: качественный диалог с пользователем - 10 баллов, обработка ошибок ввода
Код программы должен содержать комментарии для созданных функций и основных блоков

2. Добавить вызовы логгера (добавлять записи в log-файл) во все методы программы
Должна быть обязательно отражена вся вводимая и выводимая информация (а также дата, время, тип и результат выполнения операций)

Для использования программы необоходимо чтобы файл Теста находился в папке вместе с программой. Чтобы открыть отпределённых файл необходимо самостаятельно изменить переменную " filename = "?" ", заменив знак вопроса на название файла с расширением.
